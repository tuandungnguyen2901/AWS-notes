# AWS KMS (Key Management Service)

**Primary Use**: Cryptographic key management

**Key Features**:
- FIPS 140-3 Level 3 validated HSMs
- Customer Managed Keys (CMKs)
- AWS Managed Keys
- Key rotation (automatic and manual)
- Envelope encryption
- External Key Stores (XKS)
- Asymmetric keys
- Key policies and access control

**Mentioned In**: Data Protection, Key Management Service, Backup and Recovery
