# Amazon S3 (Simple Storage Service)

**Primary Use**: Object storage service

**Key Features**:
- Cross-Region Replication (CRR)
- Storage classes (Standard, IA, Glacier, etc.)
- S3 Object Lock (WORM storage)
- SSE-KMS encryption
- Gateway Endpoints (free connectivity)

**Mentioned In**: Backup and Recovery, Network Security, Data Protection, Cost Optimization
